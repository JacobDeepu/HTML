Programming Languages

  1. High Level Language
    2. A high-level language is a programming language designed to simplify computer programming.
    • General purpose languages
      • Java
      • Ruby

    • Web programming languages
      • HTML

  2. Low Level Language
    3. A low-level language is the machine-dependent language, which is near to the machine and far away from the programmer.
    • Machine language
    • Assembly language
